# codedeploy-springboot

# Pre-Requisites
    Launch Instance with some tag
    Install codedeploy-agent
# Install codedeploy-agent
    sudo yum install ruby
    wget https://aws-codedeploy-us-east-1.s3.amazonaws.com/latest/install
    chmod +x ./install
    sudo ./install auto
